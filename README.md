# Pharaoh's Call Audio Page

This repository contains a simple HTML audio landing page for **Pharaoh's Call**.

## Files

- `index.html` — the main web page
- `why-black-america-must-lead-ai.mp3` — the audio file used by the page
- `README.md` — this project description

## How to Publish with GitHub Pages

1. Upload all three files to the GitHub repository:
   - `index.html`
   - `why-black-america-must-lead-ai.mp3`
   - `README.md`
2. Commit the changes.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/root`
5. Click **Save**.

After GitHub Pages finishes publishing, the page should be available at:

`https://sairah6146.github.io/Pharaohscall/`

## Important Note

Keep `index.html` and `why-black-america-must-lead-ai.mp3` in the same folder. The audio player depends on that file location.
